Alternator: 155A (factory). ~70A DC-DC from alternator to house is a reasonable starting size.
LiFePO4 battery max charge often spec'd ~0.5C (50% of amp-hours); confirm per battery.

Chassis charging: the battery BMS protects the *cells* (voltage/current/temp); it does *not*
  replace a DC-DC (or similar) charge controller for the alternator path. Lithium can accept
  high charge current for a long time, which can overload/overheat the alternator; a DC-DC
  charger limits output current and matches charging to the house bank. Do not rely on "just
  a fuse" or the BMS alone for alternator protection.

      (e.g. >8V on L-pin per manual) or remote on/off. **Euro 5/6 caveat:** smart alternators

    **Current:** input and output limits adjustable **1--50A** (0.1A steps) -- tops out **below**
      the earlier ~70A planning note; at ~12V, continuous power is far below the 1400W nameplate
      (manual: power ~ V x I, e.g. 12V x 50A = 600W). Set input current limit to protect the
      alternator. **Fusing/cable:** manual table for ~60--70A fuse: 16mm2/6AWG if <5m, 4AWG if
      5--10m (adjust if your design uses different current).
    **Lithium:** VictronConnect battery presets; low-temperature cut-off when temp is shared
      via VE.Smart network -- **not effective if a BMS controls the charger** (per manual).
      Optional BMS control / Error 67 if BMS link lost.
    **Ground:** GND lug should make **low-impedance** contact with bare chassis metal (manual).

DC-DC install environment: Renogy/LiTime are usually **less** dust/water-protected than Orion
  XS (IP65). Mounting in a **dry cabinet** (e.g. near the coach battery disconnect) can be
  fine for those units **if** the space stays dry and you address **heat**: fanless chargers
  still rely on **convection** and manufacturer **clearance** (Orion XS manual: ~10cm around
  the unit, vertical mount, terminals down). A **stagnant** or **packed** bay will cause
  **thermal derating** (lower sustained amps) even if the electrical install is correct. If
  airflow is weak, options include passive **vent** paths for that compartment, avoiding
  burying the charger in insulation, or accepting a **premium sealed** unit for harsh exposure
  while still giving it **room to breathe**.

Solar: portable "solar suitcase" with its own charge controller; controller supports lithium
  and connects direct to the house battery. Use the controller's LiFePO4 / lithium profile
  (not lead-acid) when on lithium. Align absorption/float-style setpoints with the battery
  vendor's guidance if the controller allows custom voltages. Solar is only hooked up when neither shore nor generator is feeding the coach (generator
  takes the same inlet as shore -- mutually exclusive). Never solar at the same time as PD
  charging from shore or gen, so no overlap with the converter on the same bank.

House/chassis isolation (stock): a **simple relay with a time delay** that connects house and
  chassis batteries when energized (not a microprocessor-based isolator).
Phase 1 / alternator path (see phase1.md): remove that **direct parallel** relay path. Run
  DC-DC **input + and -** from the **chassis (starter) battery** (fused, per charger manual);
  DC-DC **output** to the house bank. All driving-time charge from the alternator to the house
  goes **only** through the DC-DC (no bypass around it). The old relay wiring is removed or
  left disconnected per install plan.
Ignition / enable: depends on charger -- Renogy D+ (DC-DC_Renogy.pdf); LiTime vehicle port;
  Victron Orion XS often uses **engine shutdown detection** in software (manual). Optional
  ignition or override later if needed.
Stock behavior (before rewire): relay closed when energized parallels banks for charging /
  boost per OEM. Shore power may also charge chassis depending on coach wiring. After rewire:
  driving path is chassis battery -> DC-DC -> house. Tradeoffs (e.g. chassis maintenance from
  shore, emergency parallel) depend on what the relay used to do -- confirm against schematics.
