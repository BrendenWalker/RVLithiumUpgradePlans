--- Phase 1 (installed / purchased) ---

PDU / shore charging: Progressive Dynamics PD4655V converter-charger (replaces WFCO 8955PEC).
  Lithium-capable; use LiFePO4 mode when house bank is lithium. Integrated BMS required on packs.
  Manual: PD4600-110197F-owners-manual.pdf — covers the **PD4655** (among others); see **page 2**
  for model coverage and that the **4655** replaces the **WFCO 8955**-class unit.
  Listing: https://www.etrailer.com/Accessories-and-Parts/Progressive-Dynamics/PD4655V.html

Alternator / DC-DC: Victron Orion XS DC-DC battery charger (Orion XS 1400 series).
  Manual: 124067-Orion_XS_DC-DC_battery_charger-pdf-en.pdf
  Input/output current limit 1--50A (0.1A steps); engine shutdown detection via VictronConnect;
  IP65; Bluetooth + VE.Direct. Set input current limit for alternator protection.
  Install / wiring detail: phase1.md, Phase 1 Wiring Diagram.md

Alternator: 155A (factory). Orion tops out at 50A -- aligns with phase 1 plan; LiFePO4 later:
  confirm charge C-rate with battery vendor.

House/chassis (stock): simple relay + time delay paralleled house and chassis when energized.
  Phase 1: relay path removed; chassis battery -> Orion -> house only (see phase1.md).

Solar: portable suitcase + own controller; Li profile when on lithium. Do not connect while on
  shore or generator (same 120V bus as PD charging).
